## 介绍
QT和OpenGL做的网格查看器，采用可编程渲染管线, 实现旋转，缩放功能。

目标是把基本的各类网格显示都实现，而且可以做到使用命令行打开。目前实现了三角形网格（obj文件），四边形网格。

## 环境
* VS 2019
* QT 5.12.3
